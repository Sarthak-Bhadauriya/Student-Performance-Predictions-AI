# Student Performance Predictor

## Overview
ML based student grade predictor (Flask app)

## Setup

1. Install dependencies  
   pip install -r requirements.txt
2. Train model  
   python train_model/train.py
3. Run app  
   python app/utils/app.py

## Usage  
- Home page for prediction  
- Bulk CSV/Excel file upload supported  
- Prediction history: /history page
